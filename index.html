Sequential vs Binary Search

How to Use

1. Masukkan Array Size terlebih dahulu (misal: 20)
2. Lalu, klik tombol Generate untuk membuat array acak
3. Masukkan Target Number yang ingin dicari (misal: 15)
4. Klik tombol Sequential Search atau Binary Search untuk memulai pencarian
5. Amati visualisasi pencarian, langkah-langkah yang diambil, serta grafik perbandingan
6. Ulangi langkah di atas dengan ukuran array dan target number yang berbeda untuk melihat perbedaan performa kedua algoritma
